# 数学优化基础
优化三要素：从**变量**的**约束条件**所限定中的集合中，找到**目标函数**极值解
优化三部曲：根据实际问题**建模**得到数学优化问题；明确**最优性条件、存在性、唯一性**等；设计优化算法
## 解
**最优解**：全局最优解，局部最优解
	最优解的必要条件：KKT条件
**近似解**：给定最小化问题$min_{x\in R^d}f(x)$和最优解$x^*=argmin_{x\in R^d}f(x)$：
	零阶定义：$f(x)-f(x^*)\leq \epsilon >0$
	一阶定义：$||\nabla f(x)||\leq \epsilon >0$
	二阶定义：||∇f(x)|| ≤ ε₁, λ_min(∇²f(x)) ≥ -ε₂，其中 ε₁, ε₂ > 0
		对于Hessian矩阵（所有二阶偏导的矩阵） ∇²f(x)，最小的特征值λ_min > 0时，函数f在x附近各个方向上均为为上开口
**算法复杂度**：给定最小化问题$min_{x\in R^d}f(x)$和最优解$x^*=argmin_{x\in R^d}f(x)$和迭代算法A；不失一般性，假定近似解衡量标准为零阶定义：
	A的**收敛率convergence rate**为r(k)：A的第k次迭代x_k满足$f(x_k)-f(x^*) ≤ O(r(k))$
	A的**迭代复杂度**为c(ε)：A的一个近似解x使得$f(x_k)-f(x^*) ≤ \epsilon$需要O(c(ε))
	总体复杂度 = 迭代复杂度 * 单步迭代时间
当f_i(x)为凸函数convex function，h_i(x)为线性函数，该优化问题为**凸优化问题**，此时KKT是最优解的充要条件。
## 凸集
**凸集**：集合中任意两个点间线段上的点也都属于该集合
**凸组合**：点$x_1...x_k$的凸组合为$\{x|x=\theta_1x_1+...+\theta_kx_k\}$, $\theta_1+...+\theta_k=1,\theta_i\ge0$（点连线构成的图形）
**凸包 convex hull**： conv S = 集合S中所有点的凸组合
**范数 Norms**：描述距离的函数，满足：
	1. 值域非负
	2. ||tx|| = |t| ||x|| for t in R
	3. ||x+y|| <= ||x|| + ||y||
中心点为$x_c$，半径为r的**范数球 Norm ball**：$\{x |\space||x-x_c||\le r\}$
**范数锥 Norm cone**：$\{(x,t) |\space||x||\le t\}$，注意这里x比之前少了一个维度给t（倒锥形）
## 凸函数
>上开口

**凸函数（零阶定义）**：对于f:R^n→R，dom ⁡f是凸集而且对所有 x, y ∈ dom⁡ f, 0≤θ≤1：f(θx+(1−θ)y) ≤ θf(x)+(1−θ)f(y)
**凸函数（一阶定义）**：对于可微函数f，$f(y)\ge f(x)^T(y-x), \forall x,y\in dom f$
**凸函数（二阶定义）**：对于二次可微函数f，$\nabla^2f(x)\ge0,\forall x \in dom f$

常见凸函数：
	指数函数，负熵xlogx，x^a(a>=1 or a<=0)
	矩阵空间：线性函数tr(A^TX)，谱范数(最大奇异值)，平方函数，最小二乘目标函数

## 平滑性与(强)凸性
二次可微函数 g: Rd→R是：
	**L-平滑**：$\forallθ∈R^d,∣eigenvalues[g′′(θ)]∣≤L$
	**μ-强凸**：$\forallθ∈R^d,eigenvalues[g′′(θ)]\ge\mu$
 **条件数** κ 定义为 κ=L/μ≥1
## 确定性平滑优化方法
>假设g为凸函数，且L-平滑
>可用收敛率和算法复杂度评价，但省略了

**下降法**：每次迭代中，函数值会严格减少一定数值，因此被称为下降法
- **梯度下降**:  $\theta_t = \theta_{t-1} - \gamma_t g'(\theta_{t-1})$ 
- **牛顿法**: $\theta_t = \theta_{t-1} - g''(\theta_{t-1})^{-1} g'(\theta_{t-1})$
**动量法**：动量法中，函数值有可能不会单调的减少
	**Nesterov's**：$θ_t = η_{t-1} - γ_t g'(η_{t-1})$ and $η_t = θ_t + δ_t (θ_t - θ_{t-1})$
	**Polyak's**：$θ_t = θ_{t-1} - αg'(θ_{t-1}) + β(θ_{t-1} - θ_{t-2})$

可用收敛率和算法复杂度来评价上述方法：
	梯度下降是一阶方法，一般是线性收敛，需要很多步但每一步便宜。
	牛顿法是二阶方法，接近最优点时常常是二次收敛（非常快），步数少，但每一步计算贵。
# 大规模监督式机器学习中的随机优化
i(t)∈{1,2,…,n}：第t次迭代时，有放回随机采样的索引
- **SGD Stochastic Gradient Descent**：$θ_t = θ_{t-1} - γ_t f'_{i(t)}(θ_{t-1})$
	基础方法
- **SAG Stochastic Average Gradient** : $\theta_t = \theta_{t-1} - \frac{\gamma}{n} \sum_{i=1}^{n} y_i^t$ with $y_i^t = \begin{cases} f_i'(\theta_{t-1}) & \text{if } i = i(t) \\ y_i^{t-1} & \text{otherwise} \end{cases}$
	可解释为“lazy”的梯度评估
	SAG 的另一种形式: $\theta_t = \theta_{t-1} - \gamma \left[ \frac{1}{n} \sum_{i=1}^{n} y_i^{t-1} + \frac{1}{n} (f_{i(t)}'(\theta_{t-1}) - y_{i(t)}^{t-1}) \right]$
	有偏更新（关于\(i(t)\)的期望不等于完整梯度(full gradient))
- **SVRG Stochastic Variance Reduced Gradient** : $\theta_t = \theta_{t-1} - \gamma \left[ \frac{1}{n} \sum_{i=1}^{n} f_i'(\tilde{\theta}) + (f_{i(t)}'(\theta_{t-1}) - f_{i(t)}'(\tilde{\theta})) \right]$
	无偏更新，两层循环，不需要存储梯度
- **SAGA** : $\theta_t = \theta_{t-1} - \gamma \left[ \frac{1}{n} \sum_{i=1}^{n} y_i^{t-1} + (f_{i(t)}'(\theta_{t-1}) - y_{i(t)}^{t-1}) \right]$
	无偏更新，一层循环，需要存储梯度
# 神经网络中的随机优化
深度学习中的随机梯度下降方法起作用的三个关键因素：Mini-batch, 动量和权重衰减
适应性随机梯度方法：使每一个参数的更新都有自己的学习率，在某些网络的训练中效果更优
	Adagrad对每个参数使用不同的学习率；
	Adadelta通过指数衰减移动平均的方式限制早期的参数对当前迭代的影响；
	RMSprop：简化Adadelta更新规则；
	Adam：适应性的调节每个参数的学习率。
改善随机梯度方法泛化性：
	权重初始化：
		基本方法：以独立同分布的高斯或者均匀分布的方式初始化所有权重
		Kaiming初始化
		Xavier初始化
	数据增强
	学习率调节：一般而言，随迭代次数的增加，学习率应该越来越小
		固定学习率
		多阶段学习率：每过一定的step数，将学习率减少一定比例
		即时变化学习率：每经过一次迭代或者epoch，以一定的方式平滑减少学习率。常用的方式有线性学习率，余弦学习率
	学习率Warmup：在训练中，在开始的几个epoch，将学习率从一个较小值线性增大到一个最大值有助于神经网络泛化
