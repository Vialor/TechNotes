# Basics
符号位： 1：负数，0： 正数
范围： `-2**31 <= num <= 2**31 - 1`
原码 true form，反码 1's complement，补码 2's complement
- 正数的反码和补码都与原码相同
- 负数的反码为该数的原码除符号位外所有位取反
- 负数的补码为该数的原码除符号位外所有位取反，然后加1

> 3 原码，反码和补码: 00000000 00000000 00000000 00000011  
> -3 原码: 10000000 00000000 00000000 00000011  
> -3 反码: 11111111 11111111 11111111 11111100  
> -3 补码: 11111111 11111111 11111111 11111101  
> 计算机用补码存储int， -num = ~num+1  
  
运算符：
&与 |或 ^异或 ~取反（非）
<< 左移（补零去溢出）>>右移（补零去溢出）
```Python
# Equivalences
num & 0b1111
num % (2**4)
num >> 4
num // (2**4)
num << 4
num * (2**4)
```
# Know-hows
```Python
x & 0 = 0; x & 1 = x: 掩码操作 num & mask 只保留 num 中与 mask 的 1 部分相对应的位
x | 0 = x; x | 1 = 1: 设值操作 num | mask 将 num 中与 mask 的 1 部分相对应的位都设置为 1
x ^ 1 = ~x; x ^ 0 = x: 翻转操作 num & mask 翻转num与mask的 1 部分相对应的位
num ^ -1 = ~num, num ^ 0 = num, num ^ num = 0 # -1表示所有位都是1，0表示所有位都是0

num&(num-1) 去除 n 的位级表示中最低的那一位 1（其余位不变）
Lowbit: num&(-num): 得到 n 的位级表示中最低的那一位 1（其余位为0）
限定位数 1s = (1<<位数) - 1, 最大位数 1s = ~0
补码 = 1s + 原码(去掉溢出位)

u32 = num & 0xFFFFFFFF # 返回数字在32位里面的储存方式，负数以补码呈现

Swap two numbers without new variables
a = a ^ b;
b = a ^ b;
a = a ^ b;

将三个数中重复的两个数去除，只留下另一个数 x ^ x ^ y
```
