# Disk Common Sense
8 bit = 1 byte, 1024 byte = 1 kilobyte(kb), 1024 kb = 1megabyte(mb)
1024 mb = 1 gigabyte(gb), 1024 gb = 1 terabyte(tb), 1024 tb = 1 petabyte(pb)
eb, zb, yb
## Solid State Drive SSD & Hard Disk Drive HHD
SSD is faster but has risks for data loss, HHD is cheaper and easy for data recovery.
SSD uses Nandflash, HHD uses hard disk.
## Portable Solid State Drives PSSD & U-drive:
They all have two chips: Nandflash 闪存, Controller 主控
PSSD usually has better Controller regarding **Wear Leveling & Garbage Collection**
U-drive is cheaper
## File System Formats
FAT: FAT16 ⇒ FAT32(U-drives) ⇒ exFAT: FAT is generally good for Nandflash and supports multiple OSes including Windows, Mac and Linux
NTFS(New Technology File System): Windows
APFS: Mac
