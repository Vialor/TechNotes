# NetFilter Hooks
PREROUTING: NF_INET_PRE_ROUTING happens before the routing decision usually when packets just arrive.
INPUT: NF_INET_LOCAL_IN happens after the routing decision has been made AND the packet is sent to the current machine.
FORWARD: NF_INET_FORWARD happens when the packets are redirected to other machines.
OUTPUT: NF_INET_LOCAL_OUT happens before sending out packets generated by the current machine.
POSTROUTING: NF_INET_POST_ROUTING is the very last before sending out packets
# iptables

| Table  | Chain                                       | Functionality                                     |
| ------ | ------------------------------------------- | ------------------------------------------------- |
| filter | INPUT FORWARD OUTPUT                        | Packet filtering                                  |
| nat    | PREROUTING INPUT OUTPUT POSTROUTING         | Modifying source or destination network addresses |
| mangle | PREROUTING INPUT FORWARD OUTPUT POSTROUTING | Packet content modification                       |
```bash
iptables -t <table> -<operation> <chain> <rule> -j <target>
         ---------- -------------------- ------ -----------
            Table          Chain          Rule     Action

// List all the rules in a table (without line number)
iptables -t nat -L -n -v

// List all the rules in a table (with line number)
iptables -t filter -L -n --line-numbers

// Delete rule No. 2 in the INPUT chain of the filter table
iptables -t filter -D INPUT 2

// Drop all the incoming packets that satisfy the <rule>
iptables -t filter -A INPUT <rule> -j DROP

iptables -A FORWARD -i eth0  -d 192.168.60.5 -p tcp --dport 23 -j ACCEPT
iptables -A FORWARD -i eth1  -s 192.168.60.5 -p tcp --sport 23 -j ACCEPT
iptables -P FORWARD DROP
iptables -A FORWARD -p tcp -s 192.168.60.0/24 -d 192.168.60.0/24 -j ACCEPT

iptables -A FORWARD -p tcp -m conntrack --ctstate ESTABLISHED,RELATED -j ACCEPT
iptables -A FORWARD -p tcp -o eth0 --dport 8080 --syn -m conntrack --ctstate NEW -j ACCEPT
iptables -P FORWARD DROP

iptables -m statistic -h
iptables -t nat -A PREROUTING -p udp --dport 8080 \
	-m statistic --mode nth --every 3 --packet 0 \
	-j DNAT --to-destination 192.168.60.5:8080
iptables -t nat -A PREROUTING -p udp --dport 8080 \
	-m statistic --mode random --probability P \
	-j DNAT --to-destination 192.168.60.5:8080
```
