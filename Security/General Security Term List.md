**Payload**: The code delivered by the exploit to be run on the victim system
	inline: delivered in one block
	staged: the first payload is just a stub which then grabs the rest of the attack code
	reverse: the payload makes the victim invite the attacker to connect (since victim sends the invitation, the attacker circumvent the firewall) 
**Encoder**: Way to obfuscate the payload so that protectors won't detect

**Trojan**: malicious code disguised as harmless program to lure user installing it
**Virus**: malicious code attached to another program
**Worm**: self replicating malicious code
