An illusion about sharing the same resources among different entities as if each entity has *fully owned the resources*.
	*resources*: **Time Sharing & Space Sharing**
	*fully owned*: Virtualization should protect private resources of each entity.
**Muti-programming**: **one** users running **multiple** programs at the same time
a technique to utilize system resources (CPU, memory, peripheral devices and etc.) efficiently by allowing other jobs to use the resources when the current job is not using them.
**Time-Sharing (Multitasking): multiple** users running **multiple** programs at the same time
CPU executes multiple jobs by switching among them so frequently that users can interact with each programing without noticing the switching.
(A time-shared OS allows many users to share the computer at the same time.)
