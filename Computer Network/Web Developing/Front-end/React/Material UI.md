# Attributes
`sx`(style): adding style to Mui components in JSX

```typescript
m: 1 // margin of 1 * 8 = 8px
mt, mb, ml, mr, mx, my // margin top, bottom, left, right, left/right, top/bottom
// similarly for padding
p, pt, pb, pl, pr, px, py
```

```typescript
xs // (extra-small): 0px
sm // (small): 600px
md // (medium): 960px
lg // (large): 1280px
xl // (extra-large): 1920px
// xs: smaller or equal than
// sm md lg xl: larger or equal than
```
# Components
Box: general purpose like `div`, can use MUI system properties
Container: centers your content horizontally
Stack: a container component for arranging elements vertically or horizontally
Grid: a two dimensional Stack
