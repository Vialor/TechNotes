A search engine is comprised of:
1. [[Web Crawler]]: A software application that automatically navigates the web by retrieving web pages and extracting links from them to discover and visit new pages.
2. Data analysis: data mining
3. An huge inverted index from terms to pages
4. Search Algorithm: provide links based on keyword
	Ranked querying strategies:
		**Term-based ranking**: based on matched terms
		**Link-based ranking**: based on web graph
		**Log-based ranking**: based on user query history

Challenges for search engines:
coverage, ranking, freshness, anti-manipulation, user load, monetization
