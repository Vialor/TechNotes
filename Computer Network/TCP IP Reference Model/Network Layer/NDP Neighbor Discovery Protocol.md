> A IPv6 protocol that replaces ARP and part of ICMPv4 in IPv4 and does more

## ICMPv6
>RS and  RA are ICMPv6 packets (type 133 and 134)

**Stateless address autoconfiguration**:
	**RS Router Solicitation**: Booting nodes broadcast RS to get RA for interface configuration
	**RA Router Advertisement**: Routers send RA periodically and on RA request to nodes in its subnet
